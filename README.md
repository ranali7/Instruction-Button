DoNotClick is a minimalist, viral web application engineered at the intersection of dark UX, reverse psychology, and variable reward systems. On the surface, it is a single webpage with a giant, pulsing red button and a strict instruction: "DO NOT CLICK." In reality, it is a highly optimized digital slot machine designed to capture attention and trigger a global curiosity gap.

The application tracks every rebellious click in real-time via a globally synced counter, serving as massive social proof that nobody follows the rules. The project leverages several established psychological triggers:

Reactance Theory: By explicitly forbidding an action, the UI creates an overwhelming urge to restore behavioral freedom by doing the exact opposite.

Intermittent Reinforcement: Because the result of a click is highly unpredictable (a fake BSOD, a Rickroll, a shrinking button), the user's brain receives dopamine spikes similar to gambling.

Social Proof & FOMO: The massive, ticking global counter proves that millions of others have already broken the rule, giving the user implicit permission to join the chaos.
